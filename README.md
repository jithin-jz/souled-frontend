## Quick Start

```bash
# Clone & install
git clone <your-repo-url>
cd Souled
npm install

# Configure environment
cp .env.example .env
# Edit .env with your API URL and Google Client ID

# Run
npm run dev
```

> App runs at `http://localhost:5173`
